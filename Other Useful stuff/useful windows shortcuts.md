# 🖥️ Useful Keyboard Shortcuts for Windows & General Use

## 🔹 Basic Windows Shortcuts
- **Ctrl + C** → Copy
- **Ctrl + V** → Paste
- **Ctrl + X** → Cut
- **Ctrl + Z** → Undo
- **Ctrl + Y** → Redo
- **Ctrl + A** → Select All
- **Ctrl + S** → Save
- **Ctrl + P** → Print
- **Alt + Tab** → Switch between open windows
- **Alt + F4** → Close current window/app
- **Win + D** → Show/Hide desktop
- **Win + L** → Lock your PC
- **Win + E** → Open File Explorer

---

## 🔹 Window & Desktop Management
- **Win + ↑** → Maximize window  
- **Win + ↓** → Minimize / Restore window  
- **Win + ← / →** → Snap window to left/right half  
- **Win + Shift + ← / →** → Move window to another monitor  
- **Win + Tab** → Task View (virtual desktops overview)  
- **Win + Ctrl + D** → Create new virtual desktop  
- **Win + Ctrl + ← / →** → Switch between virtual desktops  
- **Win + Ctrl + F4** → Close current virtual desktop  

---

## 🔹 File Explorer
- **Alt + ↑** → Go up one folder level  
- **Alt + ← / →** → Navigate backward/forward  
- **Ctrl + Shift + N** → Create new folder  
- **F2** → Rename selected item  
- **Shift + Delete** → Permanently delete (bypasses Recycle Bin)  
- **Ctrl + Mouse Scroll** → Change file/folder icon size  

---

## 🔹 Browser Shortcuts (Chrome/Edge/Firefox)
- **Ctrl + T** → New tab  
- **Ctrl + Shift + T** → Reopen last closed tab  
- **Ctrl + W** → Close current tab  
- **Ctrl + Tab / Ctrl + Shift + Tab** → Switch tabs  
- **Ctrl + N** → New window  
- **Ctrl + Shift + N** → New incognito window  
- **Ctrl + L / Alt + D** → Focus address bar  
- **Ctrl + F** → Find on page  
- **Ctrl + + / - / 0** → Zoom in / out / reset  

---

## 🔹 Text Editing
- **Ctrl + Backspace** → Delete entire word to the left  
- **Ctrl + Delete** → Delete entire word to the right  
- **Ctrl + → / ←** → Jump word by word  
- **Shift + → / ←** → Select text character by character  
- **Ctrl + Shift + → / ←** → Select text word by word  
- **Home / End** → Jump to start/end of line  
- **Ctrl + Home / End** → Jump to start/end of document  

---

## 🔹 System & Productivity
- **Win + S** → Open search  
- **Win + I** → Open Settings  
- **Win + R** → Open Run dialog  
- **Win + V** → Open Clipboard History (enable in settings first)  
- **Win + ;** → Emoji & symbol picker 😎  
- **Win + Shift + S** → Snip & Sketch (screenshot tool)  
- **PrtScn** → Capture full screen (copies to clipboard)  
- **Alt + PrtScn** → Capture active window  

---

## 🔹 Advanced Power User Shortcuts
- **Ctrl + Shift + Esc** → Open Task Manager directly  
- **Win + X** → Quick Link Menu (power tools menu)  
- **Win + Pause/Break** → Open System Properties  
- **Shift + Right-Click on File/Folder** → Extra context menu options (like *Copy as path*)  
- **Ctrl + Shift + Click App** → Run app as administrator  
- **Win + Number (1–9)** → Open taskbar apps by position  
- **Alt + Space** → Window menu (Move, Resize, Minimize, etc.)  

---

## 🔹 Special Tricks
- **Ctrl + Shift + N (in Chrome/Explorer)** → New Incognito Window / New Folder  
- **Win + . (period)** → Emoji / Kaomoji panel (ಠ‿ಠ)  
- **F11** → Toggle fullscreen in most apps/browsers  
- **Ctrl + Shift + Esc + Shift** (hold) → Open Task Manager directly (alternate)  
- **Win + H** → Voice typing (Windows speech-to-text)
